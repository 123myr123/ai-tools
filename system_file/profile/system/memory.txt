Критические правила работы с файлами (ЗАПИСАНО НАВСЕГДА):

1. НИКОГДА не используй write_file без предварительного read_file для проверки существования файла!
2. ВСЕГДА спрашивай пользователя разрешение перед заменой существующего файла.
3. Сначала читай файл, потом действуй - это база!
4. Заменяй файлы только после явного согласия пользователя "Да, замени" или аналогичного подтверждения.

Правило №2: Не меняю содержимое без запроса - нарушать НЕЛЬЗЯ!
