# AI Tools

**Это тестовый репозиторий для отработки моих навыков**

Это приложение для ИИ-ассистентов в режиме агента, основанное на библиотеке LM Studio. Приложение имеет несколько инструментов и систему профилей.ИИ не может выйти за границу папки ai-tools.

## Инструменты
* `create_file`
* `read_file`
* `write_file`
* `read_folder`
* `create_folder`
* `run_command` (по умолчанию выключен)

## Первый запуск
Если у вас не установлены библиотеки LM Studio и само приложение, перейдите на официальный сайт и установите [LM Studio](https://lmstudio.ai).

1. **Установка библиотеки LM Studio:**
   ```bash
   pip install lmstudio
   ```
2. **Запуск приложения:**
   Выполните команду для запуска `app.py`.
3. **Выбор профиля:**
   Выберите профиль `system` и начинайте работать.

## Редактирование инструментов
В папке ваших профилей рядом с файлом `prompt.txt` находится файл `tools.json`. Он отвечает за включение и выключение инструментов.

**Пример изменения настроек:**
1. В папке `ai-tools\system_file\profile\system` найдите файл `tools.json`.
2. Откройте его. Вы увидите параметр `run_command`, у которого установлено значение `0` (инструмент выключен).
3. Поменяйте это значение на `1`, чтобы включить инструмент `run_command`.

### Настройки безопасности
В корне `ai-tools` есть файл `tools_config.json`, который отвечает за параметры безопасности:
* `access` — отвечает за уровень доступа. Уровень `1` требует подтверждения пользователя при каждом вызове инструмента. Уровень `2` не требует подтверждения — инструмент выполняется автоматически.
* `ban_list` — содержит список запрещенных файлов и папок относительно директории `ai-tools`.
* `ask_list` — содержит список файлов, действия с которыми в любом случае требуют подтверждения пользователя.

## Модуль памяти (Memory)
Модуль памяти выполняет функцию сохранения контекста между сессиями. У каждого профиля есть свой файл памяти, который сохраняет данные в `memory.txt`.
