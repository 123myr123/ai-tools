# ai-tools
Это тестовый репозиторий для отработки моих навыков


# Требования
Для работы вам понадобится иметь:
     - библиотека lm studio
     - lm studio и запущенная на нем локальный сервер с ии

# Режимы
Программа имеет 2 режима обработки фото и инструменты.

# Режим с инструментами(1)
У ии есть доступ к следующим инструментам:
     - create_foldel создает папку.
     - read_folder сообщает о всех папках и файлах в указанной папке. Важно корневая папка для ии та папка где находится app.py.
     - write_file файл открывается для записи, его содержимое удаляется; если файл не существует, создается новый.
     - read_file читает файл
     - create_file создает файл, также может создать его уже с контентом

# Режим фото(2)
Позволяет отправить фото в вашу ии если она их поддерживает
